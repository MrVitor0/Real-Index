# 🚧 REAL Severity Index — MVP Roadmap

## Initial Stack

- Next.js 16 + App Router
- TypeScript strict
- Tailwind CSS v4
- shadcn/ui
- Neon PostgreSQL
- Drizzle ORM
- Upstash Redis

## Local Setup

1. Run npm install
2. Copy .env.example to .env
3. Fill the Neon and Upstash credentials
4. Run npm run dev

## Useful Commands

- npm run check
- npm run db:generate
- npm run db:push
- npm run db:studio
